Objectifs du Projet

Immersion Totale en VR

Créer une expérience de réalité virtuelle captivante où le joueur se retrouve seul dans l'espace. Le système de téléportation et d'interaction naturelle permet une navigation intuitive dans la station spatiale abandonnée.

Les interactions physiques avec les objets utilisent le système de physique d'Unity pour un réalisme optimal. Chaque geste du joueur est traduit précisément dans le jeu grâce aux contrôleurs VR.

Système de missions quotidiennes

Un système de missions quotidiennes permet au joueur de réaliser des taches que pourrait réaliser un être humain dans une station spatiale.

Le système combine des ressources trouvées dans l'environnement pour réparer des piéces du vaisseau.
